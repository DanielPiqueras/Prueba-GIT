Bienvenido a mi página de prueba

En esta página veremos información útil sobre como usar GIT

Nada más iniciar git, tendremos que dirigirnos a nuestra ruta de la carpeta o el proyecto que queramos guardar en GIT, abrirlo con GITBash, e iniciar el comando "git init"

Para ir añadiendo los archivos, se usa el comando "git add ." para añadirlos todos, aunque se pueden seleccionar uno a uno

Para comprobar que se han guardado, usaremos el comando "git status"

Para guardarlo todo, se usa el comando 'git commit -m "el mensaje de la nueva actualización"'
